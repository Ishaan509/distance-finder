# To Run project Locally:

-In Terminal:
cd distanceFinder/
npm run dev

-Open Another Terminal Window:
cd server/
node server.js

or

nodemon server.js

# To install Packages:

-In Terminal:
cd distanceFinder/
npm install

-Open Another Terminal Window:
cd server/
npm install

# If packages aldready Installed:

-In Terminal run:
npm outdated(if it shows outdated packages then run the below command)
npm update
